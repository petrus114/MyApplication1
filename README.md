# SoFutebol


Esta aplicação tem como objectivo a consulta dos planteis das equipas de futebol da primeira divisão portuguesa, classificação,
calendário e noticias. 
Consiste num sistema de login com base de dados intregada, e os nomes dos jogadores e os seus numeros da camisola, calendário, 
classificação e noticias actualizadas, irão ser obtidos através de uma página de internet da especialidade.

## Funcionamento

Necessário efectuar registo para entrar na app. No dashboard selecionamos a opção equipas, posterioremente pode selecionar a equipa
pretendida, ou filtar por zona do país. Pode seleccionar o calendário, a classificação ou as noticias.
